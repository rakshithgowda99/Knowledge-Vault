# Knowledge Vault

## Prerequisites
- Node.js
- MongoDB

## Setup

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Environment Variables**
    Create a `.env` file in the root directory to configure your database connection:
    ```env
    DATABASE_URL=mongodb://localhost:27017/knowledge_vault
    ```

3.  **Run the Application**
    ```bash
    npm run dev
    ```
